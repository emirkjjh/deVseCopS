## Система версионирования

Мы используем [Семантическое версионирование](https://semver.org/) (SemVer) для управления версиями. Формат версий следующий:

- **MAJOR** — версия изменяется, когда вносятся несовместимые изменения.
- **MINOR** — версия изменяется, когда добавляется функциональность, совместимая с предыдущими версиями.
- **PATCH** — версия изменяется, когда исправляются ошибки без изменения функциональности.
